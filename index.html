HitNet Stereo Depth Estimation

3D Object Detection using Stereo Vision

The implementation of the HitNet architecture for stereo depth estimation and 3D object detection.The implementation includes a comprehensive performance comparison with CREStereo, demonstrating HitNet's superior performance for close-range depth estimation (under 5 meters) and significantly faster execution times, making it ideal for real-time applications.

Lidar-camera Fusion

3D Object Detection using Lidar-camera Fusion

The implementation of two Lidar-camera fusion strategies- early fusion and late fusion for 3D object detection using the KITTI dataset, the PV-RCNN++ (lidar 3d object detection model) and Yolov8 (camera 2d object detection model) to evaluate each approach.

Multi-Object Tracking

2D object tracking using SORT, DeepSORT, and StrongSORT++

A comprehensive comparison of SORT, DeepSORT, and StrongSORT++ for 2D object tracking. This project analyzes their differences in implementation, tracking accuracy, speed, and robustness. SORT offers a fast and lightweight approach, while DeepSORT and StrongSORT++ incorporate deep learning to enhance tracking performance. The goal is to evaluate their strengths, limitations, and suitability for different applications.

  • Project Type: Computer Vision Research
  • Category: 2D Object Tracking
  • GitHub: View Repository
